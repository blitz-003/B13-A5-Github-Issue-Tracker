### JavaScript Quick Answers

- var is function-scoped, let is block-scoped and can change, const is block-scoped and cannot change.

- Spread operator (...) is an operator which expands arrays or objects into individual elements.

- map(), filter(), forEach()
  map returns a new array with transformed values, filter returns a new array with elements that pass a test, forEach runs a function on each element.

- Arrow function
  A shorter way to write a function, e.g., (a, b) => a + b.

- Template literals
  Strings using backticks ` ` that allow `${variables}` and multiline text with javascript logic
